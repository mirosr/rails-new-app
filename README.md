rails_new_app
=============

Description
-----------

Makes a new directory, creates a new rails app in that directory and then applies the given template. If no template name is given the default one (base) will be used.

Usage
-----

```bash
$ rails_new_app APP_NAME [TEMPLATE_NAME]
```
